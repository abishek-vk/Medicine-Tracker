Smart Medicine Expiry Tracker – Complete Flutter Application
Project Overview

Develop a complete Flutter application named Smart Medicine Expiry Tracker, a modern mobile application that helps hospitals, pharmacies, clinics, laboratories, and medical stores efficiently manage medicine inventory and monitor expiry dates.

The application should be production-quality, responsive, visually appealing, and suitable for a hackathon demonstration. Focus on clean architecture, modular code, smooth navigation, and an intuitive user experience.

Technology Stack

Develop the application using:

Flutter (Latest Stable Version)
Dart
Material Design 3
mobile_scanner
flutter_local_notifications
intl

Follow Flutter best practices and keep the project modular and scalable.
